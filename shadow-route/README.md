# 🌳 ניווט בצל — Shaded Route Navigation

אפליקציית ניווט שמוצאת מסלולי הליכה/אופניים **הכי מוצלים**, לפי גובה בניינים ומיקום השמש בזמן אמת.

## הרצה מקומית

```bash
npm install
npm run dev
```

פתח `http://localhost:3000` — **לא נדרשים מפתחות API**.

## Deploy ל-Vercel (חינמי)

1. העלה את התיקייה `shadow-route/` ל-GitHub repo נפרד
2. היכנס ל-[vercel.com](https://vercel.com) וחבר את ה-repo
3. לחץ Deploy — הכל עולה אוטומטית

## ארכיטקטורה

| שכבה | ספרייה | API |
|---|---|---|
| מפה | Leaflet + OpenStreetMap | חינמי, ללא מפתח |
| ניתוב | OSRM | חינמי, ללא מפתח |
| מיקום שמש | SunCalc.js | client-side |
| בניינים | Overpass (OpenStreetMap) | חינמי, ללא מפתח |
| חישוב גיאוגרפי | Turf.js | client-side |

## איך זה עובד

1. המשתמש בוחר מוצא + יעד
2. OSRM מחזיר עד 3 מסלולים חלופיים
3. SunCalc מחשב מיקום השמש לפי זמן + מיקום
4. Overpass מביא גבהי בניינים לאורך המסלול
5. עבור כל בניין — מחושב פוליגון צל (גובה ÷ tan(זווית שמש))
6. כל קטע מסלול מקבל ציון צל (0–1) לפי כמה מדגמים נופלים בתוך פוליגון צל
7. המסלול המוצל ביותר מוצג כברירת מחדל
