# הנחיות פרויקט

## כללי עברית ותצוגה

1. כשכותבים בעברית, אין לשלב מונח באנגלית באמצע משפט.
   מונח באנגלית מקומו בסוף המשפט, או בשורה נפרדת משלו.

2. שמות קבצים, פקודות וקטעי קוד יושבים תמיד בשורה נפרדת או בבלוק קוד,
   ולא משולבים בתוך משפט רגיל.

3. כל תוצר HTML בעברית נפתח עם התג הבא:
   ```
   <html dir="rtl" lang="he">
   ```

4. אין להציע התקנת תוסף או פאץ' שמשנה קבצים של אפליקציה אחרת.
   אם עולה צורך כזה — יש לעצור ולהסביר קודם מה הסיכון, לפני כל הצעה או ביצוע.
